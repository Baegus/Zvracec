# [Zvraceč](https://zvracec.baegus.cz)

Jednoduchá webová aplikace umožňující číst české texty pozpátku. Foneticky upraví text tak, aby při přehrávání pozpátku dával smysl. Obsahuje jednoduchý a kvalitní záznamník s možností stažení výsledné nahrávky.

Dostupné online na [Zvracec.Baegus.cz](https://zvracec.baegus.cz)

Aplikaci lze také nainstalovat a používat offline jako PWA.
